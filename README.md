# rails_custom_error_pages

Rails engine for custom error pages.


## Installation

Add this line to your application's Gemfile:

    gem 'rails_custom_error_pages'

And then execute:

    $ bundle 

Or install it yourself as:

    gem install rails_custom_error_pages
